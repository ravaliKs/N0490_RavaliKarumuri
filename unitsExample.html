Units in CSS

CSS units are used for expressing length. There are two types of Units in CSS, they are:

  1. Absolute Units
    • Millimeter mm
    • Centimeter cm
    • Pixel px
    • Points pt
  2. Relative Units
    • Relative to the size of an element em
    • Relative to the size of the root element rem
    • vw Relative to 1% of the width of the viewport*
    • vh Relative to 1% of the height of the viewport*
    • vmax Relative to 1% of viewport's* larger dimension
    • % Relative to the parent element

Min/Max width and Min/MaxHeight

The min-width/min-height property defines the minimum width/height of an element. If the content is smaller than the minimum width/height, the minimum width/height will be applied. If the content is larger than the minimum width/height, the min-width/min-height property has no effect.

The max-width/max-height property defines the maximum width/height of an element. If the content is more than the maximum width/height, it will be adjusted automatically. If the content is smaller than the max width/height, the max-width/max-height property has no effect.
