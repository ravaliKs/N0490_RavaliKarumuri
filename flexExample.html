Explain about flex and Give two different examples of flex

Flex: To position an element in HTML we use flex, to eliminate float and position properties. Inline, for text Table, for two-dimensional table data Positioned, for explicit position of an element

We have different types of properties for a flex which will allows us to position an element, they are flex-direction, flex-grow, flex-wrap, justify-content,align-items
