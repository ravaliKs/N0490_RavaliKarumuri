Explain about grid and Give two different examples of grid

Grid: Grid helps in arrangement of elements in two dimensional way in a web page without using position property. To display element in grid, we have to give the property of display to grid or in-line grid. Then elements will arrange in grid. grid-template-columns is another property that takes how many no of columns need to be present and the size of each grid element. grid-template-rows is another property that takes max- height of the element.
